# movies-dataset-analysis
Project Overview

• Exploratory data analysis of IMDB movie data
• Examines ratings, revenue, genre distribution, popularity, and other movie attributes
• Includes visualizations, trends, and insights into factors affecting movie performance

Objectives

• Understand key patterns in movie ratings and popularity
• Analyze relationships among budget, revenue, and return on investment
• Compare genre-based performance trends
• Visualize correlations and data distributions

Tools Used

• Python
• Pandas
• NumPy
• Matplotlib
• Seaborn
• Jupyter Notebook

Files in This Repository

• movies_dataset_analysis.ipynb — Main notebook
• sample_movies.csv — Sample dataset
• README.md — Documentation

Dataset Source

• Dataset originally loaded from local storage
• Sample dataset provided for demonstration

Important Note (Path Adjustment Required)

• The notebook currently loads the dataset using a local path
• Users must update this path on their own system
• Alternatively, they may use the provided sample_movies.csv

Key Insights

• Certain genres consistently receive higher ratings
• Revenue is strongly correlated with budget for blockbuster films
• Runtime, popularity, and genre show identifiable distribution patterns
• Visualizations reveal clear clusters and relationships across movie metrics

Business / Industry Interpretation

• High-budget films tend to generate higher box office returns
• Genre trends reflect audience preferences over time
• Rating vs revenue insights help understand commercial vs critical success
• Useful for production planning, marketing, and risk analysis

How to Run

• Download this repository
• Keep the sample dataset in the same folder as the notebook
• Open the notebook in Jupyter
• Update dataset path if required
• Run all cells sequentially
